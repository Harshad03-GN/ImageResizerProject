🖼️ Image Resizer Project
A simple and efficient tool for resizing images while maintaining their quality. This project is designed to provide a user-friendly interface for batch resizing images to desired dimensions, making it ideal for developers, designers, and content creators.

🚀 Features

📁 Upload and resize multiple images at once

🔄 Supports custom width and height

🧠 Maintains aspect ratio (optional)

💾 Saves resized images in desired output folder

🌐 Simple and responsive interface (if using a web UI)

🧰 Lightweight and easy to integrate into other projects


🛠️ Tech Stack

Language: Python, Azure BLOB Storage

📷 How to Use
Clone the Repository

git clone https://github.com/Harshad03-GN/ImageResizerProject.git
cd ImageResizerProject

Install Requirements

pip install -r requirements.txt

Run the Application

python app.py

Output
Resized images will be saved to the output/ directory.

You can configure output dimensions and other settings within the UI or config file.
